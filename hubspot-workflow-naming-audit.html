Proposed Naming Convention

A two-tier system: every workflow gets a functional category prefix (Tier 1) and follows consistent patterns within that category (Tier 2). This makes the full list scannable, filterable, and self-documenting.

Tier 1: Functional Category Prefixes

Prefix Purpose Est. Count Example
SYNC: DONE Property sync between objects (Contact, Company, Deal) 58 SYNC: Lead Source 2 | Contact >> Company
LFC: DONE Lifecycle stage management & lead stamping 67 LFC: Frontegg Pipeline | S2 - SQO Stamp
RTG: Routing, assignment & region logic ~15 RTG: Company Country >> Account Region
CMP: DONE Campaign & event-specific workflows 60 CMP: WI-2025-49ers Suite | AE Personal Invitation
NRT: Nurture sequences & email workflows ~15 NRT: Onboarding Nurture - Developers
ENR: Data enrichment (Clearbit, CommonRoom, HubSpot) ~5 ENR: Country | HubSpot + Clearbit
ALT: Alerts & notifications (Slack, tasks, email alerts) ~15 ALT: Slack - Deal Stuck at Stage 0
STG: Pipeline stage automations & triggers ~22 STG: Sales Pipeline - Stage 3 Tech Eval
DOP: Deal operations (renewals, ARR calc, forecasting) ~20 DOP: Create Renewal Deal
TRK: Analytics tracking (GA4, attribution, stamps) ~10 TRK: GA4 Event - MQL
AGN: Agen product-specific workflows ~10 AGN: Target Accounts Assignment
OPS: Operational / utility / misc automations ~10 OPS: Copy Record ID to New Field
FIX: Patches, temporary fixes, retro-updates ~5 FIX: Unenroll from RSA Sequence

Tier 2: Consistent Patterns Within Categories

Category Pattern Example
SYNC LIVE Always use >> for direction. Source on left, target on right. Use | to separate property from direction. SYNC: Lead Source 2 | Contact >> Company
CMP LIVE Keep channel-year code, use | separator: CMP: CODE | Action CMP: WI-2025-Referral Partner | 02 Application Approved
ALT Channel then condition: ALT: Channel - Condition ALT: Slack - Deal Stuck at Stage 1 (7+ days)
STG Pipeline abbreviation + stage: STG: Pipeline - Stage Name STG: Agen Pipeline - Stage 3 Tech Eval
ENR Property, then sources: ENR: Property | Sources ENR: Country | HubSpot + Clearbit
NRT Campaign code then audience: NRT: Code - Audience NRT: Onboarding Nurture - InfoSec
LFC LIVE Context + action with | separator: LFC: Context | Action LFC: Frontegg Pipeline | S2 - SQO Stamp

Naming Rules

+
Always Title Case after the prefix colon
+
Use >> for all data sync direction (replaces ->, >, -->, =>)
+
Use | to separate sub-components within a name
+
Keep names under ~60 characters total
x
No dates in workflow names - use the HubSpot description field
x
No personal names (Dana's, Guy, Oz) - describe the function
x
No ALL CAPS or all lowercase - stick to Title Case
x
No status markers: [TEMP], (cloned), (new), test in production names
x
No unnamed workflows - name them or disable them
+
Standardize on "Company" (not "Account") and "Deal" (not "Opportunity")
