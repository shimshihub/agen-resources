Key Takeaways

The "Agentic" Land Grab

CrowdStrike rebranded to "Agentic Security Platform," Okta leads homepage with AI agent identity, Workato pivoted to "Enterprise MCP," Prefect launched Horizon. Every major platform is rushing to claim a piece of the agent stack.

3 New Entrants to Track

Scalekit ("Auth Stack for AI") - add to Identity & Auth. Runlayer (MCP Control Plane) - competes with Agen for Work. Peta ("1Password for AI Agents") - clean MCP governance narrative.

Nobody Owns the SaaS Vendor Use Case

None of these 23 companies specifically help SaaS vendors expose their own product to external agents with per-tenant, customer-level identity and entitlements. Security players govern agents that exist. Gateway players route MCP traffic. Agen sits uniquely at enablement + governance + identity.

Bundling Threats Are Real

Microsoft (Azure APIM), CrowdStrike (Falcon), and Okta (Entra) can all bundle basic MCP governance into existing enterprise agreements. "Free" is hard to compete with even when the functionality is thinner.

Market Clustering Into Lanes

Security overlays: Zenity, Noma, Prompt, Eve, Fiddler. MCP gateways: Kong, Prefect, IBM, Microsoft, Workato. Identity: Okta, Scalekit, Teleport, Strata. Control planes: Runlayer, Airia, Obot, Peta.

Messaging Convergence

Multiple companies use nearly identical taglines: "Secure AI Everywhere" (Zenity, Noma, Prompt), "Control Plane for AI/MCP" (Fiddler, Peta, Runlayer). Differentiation is becoming harder to communicate. Agen's identity-aware, per-tenant SaaS exposure story remains distinct.
